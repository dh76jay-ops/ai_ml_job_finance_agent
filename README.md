 AI/ML Job & Finance Intelligence Agent

An intelligent AI agent that helps freshers explore the AI/ML job market in India.

 What it does??
- Searches real-time AI/ML job trends
- Provides salary insights for different roles
- Finds top hiring companies in India
- Suggests skills needed to get hired

 Tech Stack -
- Python
- smolagents
- DuckDuckGo Search
- HuggingFace Inference API
- Qwen2.5-72B-Instruct model

 How to run...
1. Clone the repository
2. Install dependencies:
   pip install smolagents duckduckgo-search
3. Add your HuggingFace token
4. Run the notebook

 Why I built this...!!
As a fresher AI/ML engineer, I wanted to understand the job market better — so I built an agent that does the research for me!
